# Gimlé

**The complete guide to building intelligent messaging agents**

Gimlé is a powerful messaging agent framework that enables you to build automated assistants across multiple platforms. Whether you're connecting to enterprise messaging systems or personal chat apps, Gimlé provides the tools to orchestrate intelligent conversations.

*Named after Gimlé, the golden hall of Norse mythology that survives Ragnarök.*
