## Requirements

- **Node.js 22+** (recommended)
- **macOS, Linux, or Windows**

## Quick Install

Install Gimlé globally using npm:

```bash
npm install -g gimli
```

Or using the install script:

```bash
curl -fsSL https://gimli.bot/install.sh | bash
```

## Verify Installation

```bash
gimli --version
```
