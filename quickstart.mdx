## Get started in three steps

Set up your first agent and connect to a messaging platform.

### Step 1: Install Gimlé

<Tabs>
  <Tab title="npm">
    ```bash
    npm install -g gimli
    ```
  </Tab>
  <Tab title="Install Script">
    ```bash
    curl -fsSL https://gimli.bot/install.sh | bash
    ```
  </Tab>
</Tabs>

Verify the installation:

```bash
gimli --version
```

### Step 2: Configure your first agent

Create an agent with your preferred AI model:

```bash
gimli agent create my-assistant
```

Configure your API key:

```bash
gimli config set anthropic.api_key YOUR_API_KEY
```

<Tip>
  Gimlé supports Claude (Anthropic), GPT-4 (OpenAI), and other models.
</Tip>

### Step 3: Connect a channel

Add a messaging channel:

```bash
gimli channels add telegram
# or
gimli channels add whatsapp
```

Start the gateway:

```bash
gimli gateway run
```

## Test your agent

Send a test message:

```bash
gimli message send --channel telegram --to @username "Hello from Gimlé!"
```

Or chat directly:

```bash
gimli agent chat my-assistant
```
